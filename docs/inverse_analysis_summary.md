### 3. 전략 개선 방향

#### ❌ 현재 전략 (롱 진입)
```
RSI < 35 (과매도) → 롱 진입
피보나치 지지선 근처 → 롱 진입
```

**문제**: 하락 추세에서 너무 일찍 진입 → 계속 하락

#### ✅ 개선된 전략 (추세 확인)
```
RSI < 35에서 반등 확인 → 롱 진입
피보나치 지지선 + 상승 캔들 → 롱 진입
볼린저 밴드 하단 + 반등 신호 → 롱 진입
```
**개선**: 바닥 확인 후 진입 → 성공률 증가

#### 🔄 또는 숏 전략 추가
```
RSI > 65 (과매수) → 숏 진입
피보나치 저항선 근처 → 숏 진입

```
